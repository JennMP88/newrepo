switchit

SwichIt is a capstone project done at Pursuit with a team of 5 members. The site mimics Clueless' (movie) Virtual Closet where users can upload an image of the tops and bottom they have in their closet. SwitchIt allows the user to browse and filter through their virtual closet if any items have been uploaded. If the closet is empty, users can upload their top or bottom to the closet. The site allows users to have an outfit randomly selected by the computer upon clicking the random button. Once the user selects an outfit their outfit can be seen in the outfit of the day page which displays a calendar of outfits worn or to be worn on a specified date.

Github Live

BikeRental

Bike Rental is a simple ecommerce frontend website. The site allows users to view different bikes they may rent according to age group. Users may view their selected items upon clicking the cart on the top of the page. Users may also remove an item as they view their total of the rental purchase.

Github

Course Catalog

Course Catalog is currently a non-mobile friendly frontend website that allows the end user to view a short description of classes that are offered. The classes are displayed in a card. If the user clicks on tab, the user can view the classes offered for that particular skill. Upon clicking on the same tab again, all of the courses appear again.
